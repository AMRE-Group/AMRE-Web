# Neighborhood hero photos

Drop a 1600×900px (or larger 16:9) JPG here named `{slug}.jpg` to replace the CSS gradient hero with a real photo. The hero will auto-detect and use the photo if present.

Required filenames:
- beverly-hills.jpg
- bel-air.jpg
- santa-monica.jpg
- culver-city.jpg
- west-adams.jpg
- mid-city.jpg

Recommended sources for free commercial-use photos:
- Unsplash (https://unsplash.com) — search "{neighborhood} Los Angeles"
- Wikimedia Commons (https://commons.wikimedia.org)
- Pexels (https://www.pexels.com)
